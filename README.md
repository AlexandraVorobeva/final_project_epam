# final_project_epam


Clone the repo:<br>
git clone https://github.com/SparklingAcidity/final_project_epam<br>
cd final_project_epam<br>


Create virtualenv:<br>

virtualenv venv<br>
source venv/bin/activate<br>
poetry install<br>


Run the sample server:<br>
poetry run app<br>


Run tests:<br>
poetry run pytest<br>
